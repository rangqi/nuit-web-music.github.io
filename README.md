# 网页音乐播放器（实验二）

功能：

- 播放 / 暂停
- 上一曲 / 下一曲
- 进度条拖动跳转
- 音量调节
- 歌名 + 歌词展示区（歌词文本可自行替换）
- 页面显示学号和姓名（在页面内直接填写占位符）

## 如何运行

1. 将陈奕迅《沙龙》的音频文件放到：`assets/shalong.mp3`
2. 将 LRC 歌词全文粘贴到 `app.js` 的 `EMBEDDED_LRC`（格式：`[mm:ss.xx]歌词`）
3. 打开 `index.html`

## 说明

- 音频文件请使用自己合法获取的版本；默认已在 `.gitignore` 忽略常见音频格式，避免误上传到公开仓库。
- 默认只有一首歌（《沙龙》），如需多首歌，编辑 `app.js` 的 `tracks` 数组即可。
